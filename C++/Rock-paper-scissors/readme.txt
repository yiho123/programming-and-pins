Problem Description
Alice and Bob are playing rock-paper-scissors game. The game rules are simple: rock crushes scissors; paper covers rock; and scissors cut paper. If both players throw the same shape, the game is tied.  Your task is to determine the number of games won by Alice and Bob, respectively.

Input Specification
•	The first line contains one integer N (1 <= N <= 100) that represents the number of games. 

•	The second line is Alice’s shape sequence. The shape sequence contains N shapes and they are separated by a space. The i–th shape in the sequence represents the shape thrown by Alice in the i–th game. There are only three shape values: “rock”, “paper”, and “scissors”.

•	The third line is Bob’s shape sequence.

Output Specification

Two integers separated by a space, representing the number of games won by Alice and the number of games won by Bob. year.
